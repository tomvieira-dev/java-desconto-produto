# Projeto — Cálculo de Desconto em Java

Este projeto é um exercício em Java focado no cálculo de descontos em produtos, utilizando conceitos básicos de orientação a objetos como classes, atributos, métodos e métodos estáticos.

## Funcionalidades

- Criar produtos com nome e preço base
- Aplicar desconto padrão (25%)
- Aplicar desconto especial (30%)
- Exibir no console o preço original e os preços com desconto
